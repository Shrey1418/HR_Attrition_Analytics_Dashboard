# HR Attrition Analytics Dashboard

An interactive HR analytics project built using SQL, Python, and Power BI to analyze employee attrition patterns, workforce behavior, and key retention factors.

---

## Project Overview

This project focuses on identifying the major causes of employee attrition using workforce analytics and interactive dashboarding techniques.

The dashboard helps HR teams and business stakeholders:
- Monitor attrition trends
- Identify high-risk departments and job roles
- Analyze employee satisfaction and overtime impact
- Understand workforce retention patterns

---

## Tech Stack

- SQL (MySQL)
- Power BI
- DAX
- Python
- Python Libraries
- IBM HR Analytics Dataset

## Key Features

- Multi-page interactive dashboard
- KPI cards for workforce overview

- Job role attrition analysis
- Overtime and salary impact analysis

---

## Key Business Insights

- Sales department showed the highest attrition rate
- Employees working overtime had significantly higher attrition
- Lower salary slabs experienced higher employee turnover
- Low job satisfaction strongly contributed to attrition
- Frequent business travel increased attrition risk
- Employees with shorter tenure were more likely to leave
